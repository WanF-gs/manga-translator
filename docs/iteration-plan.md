## 验收标准

每个 Sprint 完成后需通过：

1. **Python 语法检查**：`python3 -c "import ast; ast.parse(open(f).read())"`
2. **前端 TypeScript 检查**：`npx tsc --noEmit`（无新增错误）
3. **核心管线冒烟测试**：上传→检测→OCR→翻译→擦除→渲染→导出 全流程跑通
4. **PRD 覆盖率更新**：重新审计，确认状态变更
