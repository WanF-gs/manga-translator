# 漫画翻译系统 v3.0 完工计划

> 基于 2026-06-30 实现审计报告制定。按依赖关系排序，不设时间节点。

---

## 第一阶段：修复基础缺陷

### 1.1 数据库 Schema 补齐

- `pages` 表添加 `erase_quality_score FLOAT` 字段
- `text_regions` 表添加 `boundary_mode ENUM('rect','polygon','bezier')` 字段（默认 `rect`）
- `text_regions` 表添加 `character_id UUID` 外键关联 `characters` 表
- `pages.status` 枚举新增 `reviewing`（校对中）和 `needs_repair`（待修复）两个值
- 执行 DDL 迁移脚本，确保已有数据兼容

### 1.2 修复已知错误

- `translation-service` 的 `_calc_term_consistency()` 从返回固定值 0.85 改为真实对比术语库计算
- 前端 `OnboardingWizard` 组件的文案与实际功能对齐（不承诺尚未提供的示例项目）
- 统一 `text_regions` 表结构和 PRD 数据字典

## 第二阶段：补齐 P0 缺失功能（MVP 完整度）

### 2.1 智能擦除引擎增强（§2.4.0）

**这是 PRD 标记 P0 但当前完全未实现的最大缺口。**

- 后端 `image-service` 新增擦除质量评分模块：
  - 基于 SSIM 计算擦除区域与原背景的结构相似度
  - 实现分背景类型质量判定（网点纸/排线/渐变/纯色/复杂纹理的独立阈值）
  - SSIM < 0.85 的区域自动标记为"需人工修复"
  - 每页生成擦除质量分（0-100），写入 `pages.erase_quality_score`
- 新增 API: `GET /api/v1/pages/:pid/inpaint/quality`（Gateway 已注册路由，需后端实现）
- 前端编辑器新增：
  - 擦除质量评分展示（每页缩略图旁显示分数徽章）
  - 低质量区域红色高亮叠加层
  - 一键进入手动修补模式（跳转到对应区域）

### 2.2 检测选区视觉影响最小化（§2.2.8）

- AI 网关 `detector.py` 新增多边形轮廓输出模式（当前仅矩形）
  - 气泡内文字：使用 Canny 边缘检测 + 形态学操作提取气泡内轮廓
  - 无气泡文字：使用凸包算法生成紧密包围多边形
  - API 响应新增 `boundary_mode` 和 `points` 字段
- 后端 `detect_service.py` 接收并持久化多边形选区到 `text_regions.boundary`
- 前端 `RegionOverlay` 组件：
  - 支持多边形/贝塞尔曲线路径渲染（当前仅矩形）
  - 默认半透明 30% 显示
- 快捷键 H 一键隐藏/显示所有选区线（`useKeyboardShortcuts` + `editorStore`）

### 2.3 页面状态机完善

- 后端 `pages.status` 增加 `reviewing` → `needs_repair` 的状态流转逻辑
- 前端状态筛选器增加"待修复"筛选项（红色标签）
- 擦除质量低于阈值（SSIM < 0.85）时自动将页面状态设为 `needs_repair`

## 第三阶段：补齐 P1 缺失功能

### 3.1 内容安全与合规（模块24 — 全新）

- 后端新增 `content-safety` 服务模块（可内置于 `image-service` 或独立）：
  - 上传图片时调用第三方内容审核 API（如阿里云/腾讯云内容安全）
  - 检测色情、暴力、恐怖主义内容
  - 违规内容拦截 + 前端提示
  - 误拦截率控制在 ≤5%
- UGC 敏感词过滤：
  - 术语库公开贡献入口添加敏感词过滤
  - 角色名称/口头禅添加过滤
  - 字体名称添加过滤
- 数据隐私合规：
  - 用户数据一键导出功能（JSON 格式，含所有项目/术语/生词本/设置）
  - 用户数据一键删除功能（永久擦除）
  - 隐私政策页面完善
- 声音克隆合规：上传样本前弹出条款确认弹窗

### 3.2 新手引导体系完善（模块23 — 从存根升级）

- **示例项目**：后端预置一个 3 页示例漫画（使用无版权/原创素材）
  - 预填充检测结果、翻译结果、样式配置
  - 数据库 seed 脚本
  - 前端 `GET /api/v1/reader/sample-project` 加载
- **引导流程升级**：
  - 步骤 2 "上传漫画"改为真实文件上传区域（可拖拽测试图片）
  - 步骤 3 "一键翻译"改为调用示例项目的真实翻译结果展示
  - 引导完成后自动跳转到示例项目阅读器
- **帮助中心（/help）完善**：
  - 功能说明补充 28 个模块的简要介绍
  - FAQ 补充至少 20 个常见问题
  - 添加快捷键一览表
  - 页面级"?"帮助按钮（右下角固定悬浮，点击弹出当前页面功能说明）

### 3.3 支付系统真实化（从模拟升级）

- 接入至少一个真实支付网关（如支付宝/微信支付/Stripe）
- 订阅管理：自动续费、取消订阅、发票/收据
- 付费权益自动生效与过期处理（与 `user.plan_type` + `premium_expires` 联动）
- 前端升级/降级流程完整化（含支付确认页面）

### 3.4 字体角色绑定 + 缺字回退（模块22完善）

- 后端 `project-service` 实现：
  - `PUT /api/v1/characters/:cid/font` 绑定字体到角色
  - 翻译回填时，若文字区域关联了角色且角色绑定了字体，自动应用该字体
- 缺字回退策略：
  - 渲染时检测选中字体是否包含目标字符
  - 不支持则沿回退链查找（自定义字体 → 系统内置字体 A → 系统内置字体 B → 系统默认字体）
  - 前端编辑面板中红色标记缺字区域

### 3.5 AI 自适应排版引擎多目标优化（§2.4.3 完善）

- 实现 PRD 描述的遗传算法多目标优化：
  - 初始化种群（字号×行距×对齐方式×字间距组合）
  - 适应度函数：可读性(40%) + 美观度(30%) + 原始感(20%) + 紧凑度(10%)
  - 迭代优化 → 输出最优排版参数
- 竖排与横排混合排版支持
- 不规则气泡（星形/云形）内文字路径自适应弯曲

### 3.6 跨页上下文连贯翻译

- 后端 `translation-service` 新增：
  - 翻译前先收集同章节所有原文，提交 LLM 时携带前后文
  - 跨页连续对话自动合并为完整句子
  - 称谓/伏笔一致性检查 prompt

### 3.7 邀请链接

- 后端 `project-service` 生成一次性邀请链接（24 小时有效）
- 前端团队管理页面添加"生成邀请链接"按钮 + 复制/分享

## 第四阶段：补齐 P2 差异功能

### 4.1 端云协同推理架构（模块16 — 全新）

- 前端 ONNX.js WebAssembly 加载轻量文字检测模型（<5MB）
- Chrome 113+ WebGPU 加速推理
- 分流策略：端侧文字检测 → 云端 OCR + 翻译 + 擦除
- 端侧不可用时自动降级到云端检测
- 端侧模型版本自动同步（Service Worker 更新）

### 4.2 GPU 算力智能调度（模块21 — 全新）

- 任务优先级队列：高级版优先插队，免费版 FIFO
- 模型路由：小模型（Qwen-7B）优先处理简单文本，大模型（GPT-4V）处理复杂语境
- 批处理优化：同作品多页合并推理请求
- Redis 热缓存命中率监控
- 弹性伸缩：按需启动 GPU 节点，空闲自动缩容
- 成本监控仪表盘：GPU 使用率/费用/缓存命中率

### 4.3 主动学习闭环（模块17 — 完善）

- 自动识别低置信度样本（<70%）并加入训练候选集
- 模型微调流水线：每周自动用新增训练数据微调
- A/B 测试框架：新模型上线前对比旧模型效果
- 热更新上线（无需重启服务）
- 前端用户反馈面板显示"我的反馈帮助改进了 X 条翻译"

### 4.4 画质增强系列（模块6 — 全新）

- 漫画超分辨率：
  - 集成 waifu2x 或 Real-ESRGAN 模型
  - 网页端/line art 专用模型
  - 2K/4K 输出
- 扫描件修复：
  - 去噪点（Non-local Means）
  - 去莫尔纹（FFT 频域滤波）
  - 折痕/污渍修复（inpainting）
  - 线条断裂修复
- 黑白漫画上色：
  - 基于多模态 LLM（GPT-4V）的场景理解
  - 角色配色一致性保持
  - 分镜色调匹配

### 4.5 动态漫画真实视频合成（模块27 — 从存根升级）

- Ken Burns 运镜：AI 检测画面焦点 → 生成镜头推拉轨迹 → 关键帧插值
- 气泡弹出动画：按阅读顺序依次淡入 + 微放大
- 微动特效：前景元素轻微位移或缩放循环（2-3 帧循环）
- 震动/冲击：打击场景画面震动效果
- MP4 视频编码导出（ffmpeg 或浏览器端 WebCodecs API）
- 水印叠加（免费版） + 品牌标识
- 分享卡生成（封面 + 二维码）

### 4.6 有声剧场完善（模块26）

- 场景音效库：填充 20 种真实音频文件（战斗/环境/天气/情感/魔法等）
- 音效与对话音量独立调节
- 音效手动替换/关闭
- SRT 字幕文件生成（时间轴 + 角色标注）
- MP4 有声漫画视频导出（静态画面 + 多角色语音 + 字幕 + 音效）

### 4.7 预训练的 Python/JS SDK

- Python SDK 包（PyPI 发布）：文字检测、OCR、翻译、全流程 4 个模块
- JavaScript/TypeScript SDK 包（npm 发布）
- SDK 示例代码与文档

### 4.8 其他 P2 功能

- 多语言版本同步导出
- 网盘导入（百度网盘 OAuth / OneDrive Microsoft Graph API）
- Anki (.apkg) 导出
- 跨端接续二维码（生成 + 扫码跳转）
- PWA 离线编辑队列（IndexedDB + Background Sync）
- 无障碍 WCAG 2.1 AA 全面达标
- 社交化学习课程内容填充
- 公开词典（术语贡献 + 投票）
- 语法解析功能
- 排行榜
- 文化知识图谱可视化
- Photoshop / Clip Studio Paint 插件
